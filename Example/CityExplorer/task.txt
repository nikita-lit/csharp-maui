# Generate .NET MAUI Project: CityExplorer

Create a complete production-ready **.NET MAUI mobile application** named **CityExplorer**.

## General Requirements

Build the project using:

* **C#**
* **.NET MAUI**
* **MVVM architecture**
* **OOP principles**
* **SQLite local database**
* **Dependency Injection**
* **XAML UI**

The app must be cleanly structured, compilable, and runnable without additional manual restructuring.

---

# Project Goal

Build a smart Tallinn city guide mobile app that helps users discover attractions, save favorites, and switch application language dynamically.

The app should support:

* Browsing Tallinn attractions
* Filtering by category
* Saving/removing favorites
* Multilingual interface
* Persistent local storage

# Functional Requirements

## 1. Explore Tab

Create a tab named:

**"Avasta" / "Explore" / "РҳСҒСҒР»РөРҙРҫРІР°СӮСҢ"**

Features:

### CarouselView

Centered on page.

Display attraction cards containing:

* Image
* Name
* Short description

Must support:

* Smooth scrolling
* Swipe gestures
* Auto-scroll every few seconds

Include:

* `IndicatorView`

---

### Category Emoji Menu

Vertical emoji menu on right side.

Categories:

* рҹҸ° Historical places
* рҹҢі Parks
* рҹҚҪпёҸ Restaurants

When category is selected:

* Carousel content updates via MVVM binding

---

### Card Interaction

When user taps a card:

Show `DisplayAlert` with:

* Full attraction description
* Button:

**"Add to Favorites"**

This saves item into SQLite database.

## 2. Favorites Tab

**"Minu Lemmikud" / "Favorites" / "РҳР·РұСҖР°РҪРҪРҫРө"**

* Load favorite places from SQLite
* Display using CollectionView

Each item must have:

* Image
* Name
* Remove button

* Remove from database
* Refresh UI immediately

Handle empty database gracefully.

